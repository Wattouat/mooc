Introduction

Ce module concerne les logiciels de gestion de version

Ces logiciels permettent de conserver et de gérer le code source lors de la création d’un logiciel. Ils permettent de "versionner" et stocker un ensemble de fichiers dont souvent le code source du logiciel. Nous vous présenterons alors Github, que nous utilisons, et vous présenterons d’autres logiciels.

Petite precision:

GIT est un logiciel de gestion de versions

GitHub est un service web d'hébergement et de gestion de développement de logiciels, qui UTILISE Git
